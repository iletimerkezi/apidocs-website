# Send SMS with ASP
You can use the code block below to send bulk, single or OTP sms from your software made with ASP.

```visual-basic
Function POSTHTTPXML(address, strMessage)
    Set StrHTTP = Server.CreateObject("MSXML2.XMLHTTP")
        StrHTTP.Open "POST" , address, false
        StrHTTP.setRequestHeader "Content-type:","text/xml"
        StrHTTP.sEnd strMessage
        GETHTTPXML = StrHTTP.Responsetext
    Set StrHTTP = Nothing
End Function
```
