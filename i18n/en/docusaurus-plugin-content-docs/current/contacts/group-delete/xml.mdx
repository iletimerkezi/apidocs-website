# XML Post API
<AccessWarning />
To delete a group from your contact list using the POST method of the HTTP protocol in XML format, you can follow the instructions below.

## Request URL

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/delete-group" />

## Request Body

```xml
<request>
  <authentication>
    <key></key>
    <hash></hash>
  </authentication>
  <group>
    <id></id>
  </group>
</request>
```

### Definitions

The request sent to the server must be wrapped in the <Badge>request</Badge> tag. The definitions of its sub-tags are as follows:

<AccessDescription />

#### group
Contains information about the group to be deleted. It does not repeat. The sub-tags are as follows:

+ **id:** Contains the unique ID value of the group to be deleted.

## Server Response

```xml
<response>
  <status>
    <code></code>
    <message></message>
  </status>
</response>
```
