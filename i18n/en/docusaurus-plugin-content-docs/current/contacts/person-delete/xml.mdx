# XML Post API
<AccessWarning />
To delete a contact from your address book using the POST method of the HTTP protocol in XML format, you can follow the instructions below.

## Request URL

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/delete-contact" />

## Request Body

```xml
<request>
  <authentication>
    <key></key>
    <hash></hash>
  </authentication>
  <contact>
    <id></id>
  <contact>
</request>
```

### Definitions

The request sent to the server must be wrapped in the <Badge>request</Badge> tag. The definitions of its sub-tags are as follows:

<AccessDescription />

#### contact
Contains the information of the contact. The sub-tag is as follows.
  + **id:** Represents the unique numeric value of the contact.

## Server Response

```xml
<response>
  <status>
    <code></code>
    <message></message>
  </status>
</response>
```
