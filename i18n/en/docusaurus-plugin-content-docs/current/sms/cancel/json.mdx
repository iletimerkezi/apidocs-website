# JSON Post API
This method is used to cancel an SMS package that has been created for a future date. You can use it by sending data in JSON format using the POST method of the HTTP protocol.

:::tip Tip
Orders that have already begun delivery cannot be canceled.
:::

## Request URL

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/cancel-order/json" />

## Request Body

```json
{
  "request": {
    "authentication": {
      "key": "",
      "hash": ""
    },
    "order": {
      "id": ""
    }
  }
}
```

### Definitions

The request made to the server must be wrapped with the <Badge>request</Badge> tag. The definitions of its sub-tags are as follows:

<AccessDescription />

#### order
Contains information about the SMS order to be canceled. Its sub-tag is as follows:

+ **id:** A unique numeric value representing the order. This is the <Badge>id</Badge> value returned by the server when you make a request to send an SMS via the API. It is mandatory to send this value when making the request.

## Server Response

```json
{
  "response": {
    "status": {
      "code": "200",
      "message": "Operation successful"
    }
  }
}
```
