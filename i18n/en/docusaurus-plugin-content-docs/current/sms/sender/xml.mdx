# XML Post API
If you want to retrieve the list of approved headers defined in your account using the POST method of the HTTP protocol in XML format, you can follow the guidelines below.

## Request URL

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/get-sender" />

## Request Body (Body)

```xml
<request>
  <authentication>
    <key></key>
    <hash></hash>
  </authentication>
</request>
```

### Definations

The request to be made to the server must be wrapped in a <Badge>request</Badge> tag. The definitions of the sub-tags of this tag are as follows:

<AccessDescription />

## Server Response

```xml
<response>
  <status>
    <code></code>
    <message></message>
  </status>
  <senders>
    <sender></sender>
    <sender></sender>
    <sender></sender>
  <senders>
</response>
```

### Definations

The response from the server is always wrapped in a <Badge>response</Badge> tag. The definitions of the sub-tags of this tag are as follows:

<ResponseStatus />

#### senders
If the operation is completed successfully, it contains the information about the created order. If the operation fails, this tag is not returned. The sub-tags are as follows:

+ **sender:** Contains the approved header information.

## Sample Request
```xml
<?xml version="1.0" encoding="UTF-8" ?>
<request>
  <authentication>
    <key>demo</key>
    <hash>demo</hash>
  </authentication>
</request>
```

## Sample Successful Response
```xml
<?xml version="1.0" encoding="UTF-8"?>
<response>
  <status>
    <code>200</code>
    <message>Operation successful.</message>
  </status>
  <senders>
    <sender>eMarka</sender>
  </senders>
</response>
```

## Sample Error Response
```xml
<?xml version="1.0" encoding="UTF-8"?>
<response>
  <status>
    <code>401</code>
    <message>Membership information is incorrect.</message>
  </status>
</response>
```
