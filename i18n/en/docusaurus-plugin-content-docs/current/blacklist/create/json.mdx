# JSON Post API
If you want to add a phone number that you do not wish to send SMS to, to your blacklist by submitting it in JSON format using the POST method of the HTTP protocol, you can follow the instructions below.

## Request URL

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/add-blacklist/json" />

## Request Body

```json
{
  "request": {
    "authentication": {
      "key": "",
      "hash": ""
    },
    "blacklist": {
      "number": ""
    }
  }
}
```

### Definitions

The request sent to the server must be wrapped in the <Badge>request</Badge> tag. The definitions of its sub-tags are as follows:

<AccessDescription />

#### blacklist
Tag containing the number to be added to the list.

+ **number:** Tag containing the number to be added to the list. The <Badge>number</Badge> tag is not a repeatable tag. Only one number can be sent per request.

## Server Response

```json
{
  "response": {
    "status": {
      "code": 200,
      "message": "Successful operation"
    }
  }
}
```

The response from the server is always wrapped in the <Badge>response</Badge> tag. The definitions of its sub-tags are as follows:

## Sample Request
```json
{
  "request": {
    "authentication": {
      "key": "",
      "hash": ""
    },
    "blacklist": {
      "number": "+49172574483311"
    }
  }
}
```

## Sample Successful Response
```json
{
  "response": {
    "status": {
      "code": 200,
      "message": "Successful operation"
    }
  }
}
```

## Sample Error Response
```json
{
  "response": {
    "status": {
      "code": 466,
      "message": "Invalid number."
    }
  }
}
```
