# SMS API Entegrasyonu

:::tip İpucu
SMS API servisimizi kullanabilmeniz için üye girişi yaptıktan sonra Ayarlar / Güvenlik / Erişim İzinleri sayfamızdaki "API kullanımına izin ver." seçeneğini aktifleştirmelisiniz. https://panel.iletimerkezi.com/settings/security/access
:::

API, yazılımların birbiri ile sorunsuz ve hızlı bir şekilde konuşmasını sağlayan yapılardır.
API'ler her yazılım dili ile uyumlu olarak çalışırlar. İleti Merkezi,
**toplu SMS API**, **tekli SMS API** veya **OTP SMS API** özellikleri ile,
kendi yazılımlarınıza ihtiyacınıza yönelik SMS gönderme özellikleri eklemenizi sağlar.

Uygulamalarınıza kısa zamanda, kolayca entegre edebileceğiniz,
stabil çalışan, yüksek kapasiteli SMS API‘miz ile kendi ürünlerinizden
SMS gönderimi yapabilirsiniz. Birebir mesajlaşma, alarm sistemleri,
bilgilendirme & aktivasyon vb. servisler için kolayca kulanabilirsiniz.

Platformumuza ücretsiz üye olan tüm müşterilerimizin hesaplarına,
sistemin performansını ve detaylı SMS gönderim raporlarımızı test etmeleri için
**[100 adet SMS kredisini ücretsiz olarak tanımlıyoruz](https://www.iletimerkezi.com/panel/auth/signup?ref=toplusmsapi.com)**.
