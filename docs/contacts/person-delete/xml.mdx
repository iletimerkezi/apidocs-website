# XML Post API
<AccessWarning />
Rehberinizdeki bir kişiyi, HTTP protokolünün POST metodunu kullanarak, XML formatında silmek için, aşağıdaki yönergeleri takip edebilirsiniz.

## İstek yapılacak adres

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/delete-contact" />

## İstek İçeriği (Body)

```xml
<request>
  <authentication>
    <key></key>
    <hash></hash>
  </authentication>
  <contact>
    <id></id>
  <contact>
</request>
```

### Tanımlar

Sunucuya yapılacak olan istek <Badge>request</Badge> etiketi ile sarmalanmalıdır. Bu etiketin alt etiketlerinin tanımları aşağıdaki gibidir:

<AccessDescription />

#### contact
Kişi bilgisini içerir. Alt etiketi aşağıdaki gibidir.
  + **id:** Kişiyi temsil eden benzersiz nümerik değeri ifade eder.

## Sunucu yanıtı

```xml
<response>
  <status>
    <code></code>
    <message></message>
  </status>
</response>
```

### Tanımlar

Sunucudan gelen yanıt her zaman <Badge>response</Badge> etiketi ile sarmalanmıştır. Bu etiketin alt etiketlerinin tanımı aşağıdaki gibidir:

<ResponseStatus />
