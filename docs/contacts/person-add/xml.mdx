# XML Post API
<AccessWarning />
Rehberinize, HTTP protokolünün POST metodunu kullanarak, XML formatında kişi eklemek için, aşağıdaki yönergeleri takip edebilirsiniz.

## İstek yapılacak adres

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/add-contact" />

## İstek İçeriği (Body)

```xml
<request>
  <authentication>
    <key></key>
    <hash></hash>
  </authentication>
  <contact>
    <name><![CDATA[...]]></name>
    <surname><![CDATA[...]]></surname>
    <email><![CDATA[...]]></email>
    <gsm></gsm>
    <custom_fields>
      <custom_field>
        <id></id>
        <value></value>
      </custom_field>
    </custom_fields>
    <groups>
      <id></id>
      <id></id>
    </groups>
  </contact>
</request>
```

### Tanımlar

Sunucuya yapılacak olan istek <Badge>request</Badge> etiketi ile sarmalanmalıdır. Bu etiketin alt etiketlerinin tanımları aşağıdaki gibidir:

<AccessDescription />

#### contact
Eklenen kişi bilgisini içerir. Alt etiketleri aşağıdaki gibidir.

  + **name:** Kişi ismini ifade eder.
  + **surname:** Kişinin soyismini ifade eder.
  + **email:** Kişinin eposta adresini ifade eder.
  + **gsm:** Kişinin cep telefonu bilgisini ifade eder.

  + #### custom_fields
  Kişi için tanımlanmış rehber alanlarını ifade eder. Alt etiketleri aşağıdaki gibidir.

    + #### custom_field
    Kişi için tanımlanmış rehber alanını ifade eder. Tanımlanan alan kadar tekrar eder. Alt etiketleri aşağıdaki gibidir.

      + **id:** Rehber alanını temsil eden benzersiz nümerik değer.
      + **value:** Rehber alanında kişi için tutulan değeri ifade eder.

  + #### groups
  Kişinin dahil olduğu grupları temsil eden benzersiz nümerik değerleri içerir.

## Sunucu yanıtı

```xml
<response>
  <status>
    <code></code>
    <message></message>
  </status>
  <contact>
    <id>1007</id>
    <name></name>
    <surname/>
    <email/>
    <gsm></gsm>
    <custom_fields>
      <custom_field>
        <id>1</id>
        <value>27/09/2024</value>
      </custom_field>
      <custom_field>
        <id>2</id>
        <value>cip</value>
      </custom_field>
    </custom_fields>
    <groups>
      <id>7</id>
      <id>12</id>
    </groups>
  </contact>
</response>
```
