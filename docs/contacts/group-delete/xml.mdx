# XML Post API
<AccessWarning />
Rehberinizdeki bir grubu, HTTP protokolünün POST methodunu kullanarak, XML formatında silmek için, aşağıdaki yönergeleri takip edebilirsiniz.

## İstek yapılacak adres

<RequestURL type="POST" url="https://api.iletimerkezi.com/v1/delete-group" />

## İstek İçeriği (Body)

```xml
<request>
  <authentication>
    <key></key>
    <hash></hash>
  </authentication>
  <group>
    <id></id>
  </group>
</request>
```

### Tanımlar

Sunucuya yapılacak olan istek <Badge>request</Badge> etiketi ile sarmalanmalıdır. Bu etiketin alt etiketlerinin tanımları aşağıdaki gibidir:

<AccessDescription />

#### group
Silinecek grup ile ilgili bilgileri içerir. Tekrar etmez. Alt etiketleri aşağıdaki gibidir.

+ **id:** Silinecek grubun benzersiz id değerini içerir.

## Sunucu yanıtı

```xml
<response>
  <status>
    <code></code>
    <message></message>
  </status>
</response>
```

### Tanımlar

Sunucudan gelen yanıt her zaman <Badge>response</Badge> etiketi ile sarmalanmıştır. Bu etiketin alt etiketlerinin tanımı aşağıdaki gibidir:

<ResponseStatus />
